Case Study / Lead Gen · Messaging

Facebook Marketplace AI Outreach

Scrapes marketplace listings, uses an LLM to qualify each one and draft a tailored message, then reaches out over WhatsApp automatically — turning raw listings into conversations.

01Overview

A lead-generation pipeline that runs an Apify actor over marketplace listings, loops each result through an LLM that qualifies it and drafts a personalized WhatsApp message, sends it, and records the outcome — plus an upload/HTTP step to store artifacts.

02Business problem

Manually trawling marketplace listings and messaging sellers is slow and inconsistent. The team wanted qualified listings turned into personalized outreach automatically.

06AI components

  • A Basic LLM Chain on AWS Bedrock with a Structured Output Parser qualifies each listing and drafts a tailored message as structured output.

07Integrations

  • Apify — marketplace scraping actor.
  • AWS Bedrock — qualification + drafting.
  • WhatsApp — outreach messages.
  • Google Sheets — row tracking.
  • An HTTP/upload step for artifacts.

08Technical challenges

  • Normalizing messy scraped listing data before the LLM.
  • Keeping messages personalized yet compliant.
  • Looping per-item without hitting messaging limits.
  • Conditional (If) routing so only qualified listings get messaged.

09Reliability engineering

  • Deployment: self-hosted n8n, Apify-actor-driven batch.
  • Monitoring: per-row status in Sheets.
  • Error handling: Loop-Over-Items with conditional routing; an If gate skips unqualified rows.
  • Validation: Structured Output Parser enforces a consistent message schema.
  • Scalability: batch scrape + per-item loop scales with the dataset.

11Lessons learned

Putting a Structured Output Parser after the LLM made the drafted messages consistent enough to send unattended — structure is what makes generative output operationally safe.

12Future improvements

  • Add reply handling + a follow-up sequence.
  • Dedup sellers across runs.
  • Add a compliance/rate guard per number.
