Candidate Feedback Analysis

Captures onboarding feedback, runs it through an LLM with a structured-output parser, and generates a personalized follow-up email — turning raw form submissions into tailored outreach.

01Overview

A two-part flow: capture and analyze feedback submissions with an LLM (writing a structured result back to the sheet and CRM), then, on a delay, generate and send a personalized follow-up email from the analyzed data.

02Business problem

Onboarding feedback piled up unread, and generic follow-ups felt impersonal. The team wanted each submission analyzed and answered with a tailored message — automatically, shortly after submission.

06AI components

  • A Basic LLM Chain on AWS Bedrock analyzes the feedback; a Structured Output Parser enforces a consistent shape so the result can drive an email template reliably.

07Integrations

  • Webhooks — form submission.
  • Google Sheets — store + state.
  • AWS Bedrock — analysis + drafting.
  • Gmail — delayed personalized email.
  • CRM upsert — HubSpot.

08Technical challenges

  • Enforcing a consistent LLM output shape for templating (Structured Output Parser).
  • A delayed-send gate — email only sent after the analysis exists and a time threshold passes.
  • Dedup so a submission is analyzed and emailed once.

09Reliability engineering

  • Deployment: webhook capture + scheduled delayed sender.
  • Monitoring: per-row status (analyzed / emailed) in Sheets.
  • Error handling: IF gates require the AI result before sending; not-yet-sent guard prevents duplicates.
  • Validation: Structured Output Parser guarantees the email template's fields exist.
  • Scalability: capture and send are decoupled, so bursts of submissions don't block delivery.

11Lessons learned

A Structured Output Parser is what makes LLM output safe to drop into an email template — without it, personalization breaks the moment the model phrases things differently.

12Future improvements

  • Sentiment scoring to prioritize at-risk feedback.
  • Route negative feedback to a human.
  • A/B test follow-up templates.
